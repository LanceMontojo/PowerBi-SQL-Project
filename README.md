# Overview

This is an independent project I put together to sharpen my data analysis skills using Power BI. Because finding a clean dataset in the real world is rare, I’m using the [Retail Store Sales: Dirty for Data Cleaning](https://www.kaggle.com/datasets/ahmedmohamed2003/retail-store-sales-dirty-for-data-cleaning). This gives me a great opportunity to dive deep into the full ETL (Extract, Load, Transform) process and tackle messy, real-world data cleaning challenges before building out the dashboards.

## Challenges
- How much revenue are we making?
- Which products sell the most?
- Which categories generate the most revenue?
- Which locations perform the best?
- Which payment methods are customers using?
- Does giving discounts increase sales?
- Which months perform the best?
